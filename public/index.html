TP-Link Smart Plug Controller

Control your TP-Link HS100 smart plugs via simple HTTP requests. Perfect for creating desktop shortcuts!

API Endpoints

GET /toggle?host=<ip>

⭐ Toggle plug power state - Turns ON if currently OFF, or OFF if currently ON. Perfect for desktop shortcuts!

http://localhost:3000/toggle?host=192.168.1.133
GET /on?host=<ip>

Turn a smart plug ON

http://localhost:3000/on?host=192.168.1.133
GET /off?host=<ip>

Turn a smart plug OFF

http://localhost:3000/off?host=192.168.1.133
GET /status?host=<ip>

Get plug information and current power state

http://localhost:3000/status?host=192.168.1.133
Note: Replace 192.168.1.133 with your smart plug's actual IP address.

Example Response

Toggle success response:

{ "success": true, "host": "192.168.1.133", "previousState": "off", "newState": "on", "message": "Plug toggled on successfully" }

On/Off success response:

{ "success": true, "host": "192.168.1.133", "state": "on", "message": "Plug turned on successfully" }

Error response:

{ "error": "Failed to control plug", "details": "Connection timeout", "host": "192.168.1.133" }

Creating Desktop Shortcuts

Control your lamp with a single double-click:

Option 1: Toggle (Recommended)

One shortcut that automatically turns your lamp on or off:

  1. Right-click on your desktop and select "New → Shortcut"
  2. For the location, enter:
    "C:\Program Files\Mozilla Firefox\firefox.exe" "http://localhost:3000/toggle?host=192.168.1.133"
    (Replace with your browser path and plug IP address)
  3. Name it "Toggle Lamp"
  4. Set a custom icon if desired

Option 2: Separate On/Off

Create two separate shortcuts for explicit control:

  1. Follow the steps above, but use /on for one shortcut and /off for another
  2. Name them "Turn Lamp On" and "Turn Lamp Off"

Finding Your Plug's IP Address

  1. Run the discovery script: node example.js discover
  2. Check your router's DHCP client list
  3. Use the TP-Link Kasa mobile app to view device information
Tip: Set a static IP for your smart plugs in your router settings to prevent the IP from changing!
